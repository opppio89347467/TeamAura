# 풀이

- 전체 쌍 개수는 N개 수에서 서로 다른 인덱스 두 개를 고르는 것이므로 N(N-1)/2이다.
- 우리가 직접 gcd(a[i], a[j]) == 1을 검사하면 O(N²)라서 크면 터지므로, 정수론으로 한 번에 센다.
- 각 d(1 ≤ d ≤ maxA)에 대해 “d의 배수인 수가 몇 개 있는지” 개수 cnt[d]를 구한다.
- 이때 gcd(a[i], a[j]) == 1인 쌍의 개수는 ∑(d=1..maxA) μ(d) * C(cnt[d], 2) 로 계산할 수 있다(μ는 뫼비우스 함수).
- 직관적으로, “d의 배수인 수들의 쌍”을 모두 세어 놓고, 뫼비우스 포함·배제로 gcd가 정확히 1인 쌍만 남기는 과정이다.
- μ(d)는 에라토스테네스/선형체로 maxA까지 한 번에 전처리하고, cnt[d]는 값 배열을 한 번 훑으면서 각 값의 약수(혹은 배수) 개수를 누적해 구한다.
- 최종적으로 answer = ∑ μ(d) * cnt[d] * (cnt[d]-1) / 2 를 계산하면 gcd=1인 쌍의 총 개수가 된다.
